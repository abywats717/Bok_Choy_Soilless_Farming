# Bok_Choy_Soilless_Farming
Code used in the APC and microbiota analysis of nutrient solution sampled from five different soilless farming systems.

apc_analysis - the code for the APC analysis:
hydro_dada2 - dada2, PCA, PERMANOVA:
hydro_differential_abundance - ALDEx2:
hydro_shannon - Shannon diversity analysis:
Microbial_community_analysis - Occupancy abundance, heat map, upset plot:
ASV_all - the asv file:
apc_data - data for sample APC:
metadata - metadata for all samples:
metadata_updated - metadada for samples after filtering
