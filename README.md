# Bok_Choy_Soilless_Farming
Code used in the APC and microbiota analysis of nutrient solution sampled from five different soilless farming systems.

## Repository structure

- **apc_analysis** – code for the APC analysis  
- **hydro_dada2** – DADA2 processing, PCA, PERMANOVA  
- **hydro_differential_abundance** – ALDEx2 analysis  
- **hydro_shannon** – Shannon diversity analysis  
- **Microbial_community_analysis** – occupancy–abundance, heatmaps, upset plots  
- **ASV_all** – ASV table  
- **apc_data** – APC sample data  
- **metadata** – metadata for all samples  
- **metadata_updated** – metadata after filtering
